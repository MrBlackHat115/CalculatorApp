# Advanced CalculatorApp (C#)

A C# Windows Forms scientific calculator that performs both basic and advanced mathematical operations.

## Features
- Two-number input with result display  
- Min, Max, Abs, Sign  
- Trigonometric functions: Sin, Cos, Tan  
- Power, Square Root, Log, Log10, Exp  
- Clear, Reset, and Copy First Number  
- Input validation and error handling  

## Tech Stack
- Language: C#  
- Framework: .NET Windows Forms  
- IDE: Visual Studio  
- Library: System.Math  

## How to Run
- Open the project in Visual Studio  
- Build the solution  
- Press F5 to run
